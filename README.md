## Insructions

1. `git clone https://github.com/avaldivi/ui-mail.git`
2. `cd ui-mail/`
3. `npm install`
4. `npm start` to run project
5. `npm test` to run test

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
